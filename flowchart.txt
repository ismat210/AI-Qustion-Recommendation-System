🧠 SIMPLE MENTAL MODEL (VERY IMPORTANT)

Think of your system as 5 layers:

🟢 1. UI Layer

Streamlit → user interacts

🔵 2. API Layer

FastAPI → receives requests

🟡 3. Processing Layer

OCR + NLP → extract text & structure

🟠 4. Intelligence Layer

ML → recommendation engine

🔴 5. Infrastructure Layer

Docker + Kubernetes + Logging + Tests




UPLOAD
  ↓
save file → backend/data/raw/

PROCESS
  ↓
extract + clean + NLP
  ↓
save JSON → backend/data/processed/

TRAIN ML
  ↓
train model
  ↓
save model → backend/data/models/

RECOMMEND
  ↓
load latest model from backend/data/models/
